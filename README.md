# oss-audit-24BCE10654

* This is my Open Source Software lab project for auditing **Python** on **Linux (Ubuntu environment)**.

* I’m **Syed Arham Ali** (Reg no: 24BCE10654). In this project, I created a few shell scripts to display system information, check package details, audit directories, and analyze log files.

* The scripts included in this repository are:

  * System Identity Report
  * FOSS Package Inspector
  * Disk and Permission Auditor
  * Log File Analyzer
  * Open Source Manifesto Generator

* These scripts demonstrate basic Linux concepts such as variables, loops, conditionals, and file handling.

* All scripts are written in Bash and can be executed in a Linux environment.
